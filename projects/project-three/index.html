Sticky Fingers - Robotic Gripper for Shape Recognition

A sensor-equipped robotic gripper designed for precise shape recognition and manipulation.

Introduction

Our robotic gripper utilizes a dynamic rack-and-pinion mechanism inspired by state-of-the-art designs to control finger spacing dynamically. This allows effective gripping, rolling, and shape recognition of objects like spheres and cubes through extensive sensor data collection.

Design and Fabrication

  • Lightweight Modular Fingers: Independently 3D-printed fingers for easy customization and iteration.
  • High-Friction Surfaces: Applied anti-slip material to ensure secure grasping on the finger without the sensors.
  • Rack-and-Pinion Mechanism: Allows precise and parallel adjustment of the finger distance (Eventhough our gripper has this mechanism we didn’t have the time to test it with it, but it would certainly improve the grasping of more complex objects).

Data Acquisition and Processing

Sensor data from three force-sensing resistors (FSRs) was systematically collected during controlled object-rolling experiments:

  • Timestamped Voltage Data: Captured detailed temporal force data.
  • Data Parsing and Transformation: Structured raw readings into a consistent format for neural networks.
  • Feature Engineering: Calculated the time interval (delta_t) between readings to capture rolling dynamics.
  • Automated Batch Processing: Created a scalable pipeline converting raw data to structured .npy files for machine learning.

Machine Learning Implementation

We used a Long Short-Term Memory (LSTM) neural network for classifying the rolled objects:

  • Network Architecture:

    • Two-layer LSTM with 16 hidden units each, effectively capturing temporal dependencies.
    • Dense layers and dropout regularization to enhance model performance.
    • Softmax output for classifying objects into spheres or cubes.

  • Training and Validation:

    • Data split into training (30%) and testing (70%) sets.
    • Class balancing through computed weights.
    • Early stopping to prevent overfitting, optimized via the Adam optimizer.

Results

  • Confusion Matrix: Confusion matrix with 94% percent on categorizing spheres and 86% on squares
  • Training and Testing Loss: We obtained a decreasing training and testing loss as the epoch increased

Key Takeaways

Our project showcases an efficient, modular approach to robotic gripper design capable of high-accuracy shape recognition through sensor-driven data acquisition and deep learning methodologies.
